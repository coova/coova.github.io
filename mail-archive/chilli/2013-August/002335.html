After you build using dhbulid, do you have a debian/tmp or a
debian/coova-chilli directory? Some deb tools seems to use one or the
other. Try changing:

        $(MAKE) DESTDIR=$(CURDIR)/debian/tmp install

to be 

        $(MAKE) DESTDIR=$(CURDIR)/debian/coova-chilli install

in debian/rules.

Actually, you can use the option:

      --ethers=STRING           File containing a mapping of MAC
addresses to  static IP addresses

In the typical DHCP server ethers file format. This will not, however,
authenticate the session. You can also use macauth and return a
Framed-IP-Address to set the IP of connecting clients.

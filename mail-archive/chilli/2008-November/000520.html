Hi Guys
The reason I've posted up a dummy main.conf on my site for use with the 
freebsd port is because the conf files bundles with coova as standard 
are intended to be used with init.d which will dynamically generate the 
config files for you through the values defined in /etc/chilli/config, 
this is all well & good but FreeBSD does not use init.d, it uses rcng 
instead.

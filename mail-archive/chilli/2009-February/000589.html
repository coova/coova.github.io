I have been running Linux since 1998. Files don't change by themselves 
unless they are designed to do that. Chilli is not designed to change 
itself so you need to look on your system and find out what has 
changed.   Chilli is quite complex, lots of moving parts lots of places 
for it to break. We keep everything in the config file and for the last 
4 years(?) chilli has never changed. We actually still use pretty much 
the same files to run it as we did when we first setup.

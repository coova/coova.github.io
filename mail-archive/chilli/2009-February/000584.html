My preferred way to do a kiosk doesn't use chilli. Rather, to use
XULRunner on ubuntu (or any Linux). Create a XULRunner application, full
screen, and launched from the guest user X session startup script. If
you are using chilli, you might control several 'kiosks' from one router
box.

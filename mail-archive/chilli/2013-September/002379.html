Hello Mike,

You don't say if you are using the built-in web server, or another server
software. (nginx, Apache, ...)

CoovaChilli's built-in web server supports a very basic CGI-style interface.

We use a central server for managing the UAM for each client, but I once
did a splash page in Python.

I think you can make a PHP script, but I recommend to use a separate web
server and set-up there your UAM.

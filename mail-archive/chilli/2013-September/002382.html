I don't think it can be disabled, but doesn't matter.

To enable Apache, you only have to set-up Apache and configure the
following parameters of CoovaChilli accordingly:

* HS_UAMSERVER: This is the server address. If you have Apache in the same
machine as Coovachilli, it should be $HS_UAMLISTEN (e.g. 'uam.example.com')
* HS_UAMFORMAT: Full URL to the UAM script. (e.g. '
https://uam.example.com/login.php')

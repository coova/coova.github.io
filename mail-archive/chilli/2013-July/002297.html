Better would be to use DHCP relay options, 

      --dhcpgateway=STRING      DHCP gateway addresss for relay
      --dhcpgatewayport=INT     DHCP gateway port for relay
(default=`67')
      --dhcprelayagent=STRING   DHCP relay agent IP addresss (default 

So that chilli can learn the IPs as they are assigned...
